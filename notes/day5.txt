* git remote / origin
	
	-리모트 저장소 확인 -> 로컬 저장소와 연결된 리모트 저장소 확인 => $git remote -v 
		ex) C:\Users\jckim\git\bitcamp-study>git remote -v
		     origin  https://github.com/kimjoochang/bitcamp-study.git (fetch) -> fetch = 가져오는 곳
		     origin  https://github.com/kimjoochang/bitcamp-study.git (push) -> push = 올리는 곳

	-리모트 저장소 추가하기 -> git remote add <리모트저장소 이름> <url>

	- remote : 원격저장소를 지칭

	- origin : 저장소 생성 시 기본 이름 ()
